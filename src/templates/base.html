Como consultar os dados?

Para consultar dados isolados, vá para a página "Consultar", selecione a opção que desejar em cada uma das caixas de seleção e clique na lupa à direita.

Para iniciar uma nova pesquisa, clique no botão de atualizar, a setinha curvada, à direita da lupa.

Como comparar dados?

Para comparar dados vá para a página "Consultar", selecione uma ou mais opções em cada uma das caixas de seleção e clique na lupa à direita.

Para iniciar uma nova pesquisa, clique no botão de atualizar, a setinha curvada, à direita da lupa.
