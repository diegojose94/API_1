  • Como consultar dados?




    Para consultar dados isolados, vá para a página "Consultar", selecione apenas uma opção em cada uma das caixas de seleção e clique na lupa à direita.

    Para iniciar nova pesquisa, clique no botão atualizar.



    Como comparar dados?


    Para comparar dados, vá para a página "Consultar", selecione uma ou mais opções em cada uma das caixas de seleção e clique na lupa à direita.

    Para iniciar nova pesquisa, clique no botão atualizar
